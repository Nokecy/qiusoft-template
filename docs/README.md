## 📚 组件文档

### SmartNavigation 智能导航组件

专为 UmiJS KeepAlive 多tab环境设计的智能导航组件，解决多tab项目中的页面切换和数据刷新问题。

**文档列表：**

- 📖 [使用文档](./smartNavigation使用文档.md) - 完整的使用指南，包含示例和最佳实践
- 📋 [API参考](./smartNavigation-API参考.md) - 详细的API文档和类型定义
- 🚀 [快速参考](../src/components/smartNavigation.README.md) - 组件目录下的快速参考文档
- 💡 [示例代码](../src/components/examples/SmartNavigationExample.tsx) - 完整的使用示例

**核心特性：**
- 🎯 智能tab检测，避免重复创建
- ⚡ 参数冲突检测，提示用户确认
- 🔄 KeepAlive缓存自动更新
- 🎨 自定义确认对话框
- 🐛 内置调试模式
