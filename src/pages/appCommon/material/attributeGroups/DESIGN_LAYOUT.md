## 颜色和主题设计

### 主色调
- **主要颜色**: `#1890ff` (蓝色) - 按钮、链接
- **成功颜色**: `#52c41a` (绿色) - 成功状态
- **警告颜色**: `#faad14` (橙色) - 警告提示  
- **错误颜色**: `#f5222d` (红色) - 错误状态、删除操作
- **中性颜色**: `#8c8c8c` (灰色) - 辅助文本

### 数据类型标签颜色
- **文本类型**: `#108ee9` (蓝色)
- **数字类型**: `#87d068` (绿色)  
- **布尔类型**: `#f50` (红色)
- **日期类型**: `#722ed1` (紫色)

### 状态指示
- **已选择**: 背景 `#f6ffed`, 边框 `#b7eb8f`, 文字 `#52c41a`
- **已存在**: 背景 `#fff7e6`, 边框 `#ffd591`, 文字 `#fa8c16`
- **禁用状态**: 背景 `#f5f5f5`, 文字 `#bfbfbf`

## 交互状态设计

### 按钮状态
```
[主要按钮]     [次要按钮]     [危险按钮]
正常: 蓝色      正常: 白色      正常: 红色
悬停: 深蓝      悬停: 浅灰      悬停: 深红
点击: 更深蓝    点击: 灰色      点击: 更深红
禁用: 浅蓝      禁用: 浅灰      禁用: 浅红
```

### 表格行状态
```
未选择: 白色背景
悬停: #fafafa 背景
选中: #e6f7ff 背景，左边框蓝色
```

### 加载状态
```
骨架屏: 波浪式加载动画
按钮加载: 转圈图标 + 文字
表格加载: 中央转圈指示器
```

## 间距系统

### 基础间距
- **xs**: 4px - 小元素间距
- **sm**: 8px - 相关元素间距  
- **md**: 16px - 组件间距
- **lg**: 24px - 区域间距
- **xl**: 32px - 大区域间距

### 应用示例
```scss
.pageHeader {
  margin-bottom: 24px; // lg间距
}

.infoCard {
  margin-bottom: 24px; // lg间距
  padding: 20px 24px;  // 内边距
}

.buttonGroup {
  gap: 8px; // sm间距
}
```

## 字体系统

### 标题层级
- **h1**: 32px, 粗体 - 页面主标题
- **h2**: 24px, 粗体 - 区域标题  
- **h3**: 20px, 中等 - 卡片标题
- **h4**: 16px, 中等 - 小标题

### 正文字体
- **大正文**: 16px - 重要内容
- **正文**: 14px - 常规内容
- **小正文**: 12px - 辅助信息

### 字体权重
- **粗体**: 600 - 标题、重要数据
- **中等**: 500 - 标签、按钮文字
- **正常**: 400 - 常规文本

这个布局设计确保了：

- ✅ 操作流程直观
- ✅ 视觉一致性强
- ✅ 响应式适配良好
- ✅ 无障碍访问友好
