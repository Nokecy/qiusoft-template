# 项目管理可操控原型（方案 2：docs 静态页）

## 如何打开

- 直接用浏览器打开：`docs/prototypes/project-management/index.html`
- 本原型为纯静态页面，不依赖后端接口，使用 mock 数据演示交互。

## 包含的交互点

- 里程碑拖拽连线：拖拽节点位置；开启“连线模式”后，从节点右侧圆点拖拽到目标节点以建立父子关系（父 → 子，写入 `parentCodes`）。
- 项目详情：任务/问题/风险列表支持“查看详情”（弹窗）与“编辑入口”（弹窗模拟跳转到 `/form?id=...`）。
- 我的工作区：补齐 Tabs（我的任务/我的问题/我的风险），用 mock 表格展示。
- 我的关注：支持“取消关注”，问题关注补齐 `requiresApproval` 字段。

## 说明

- 这是用于产品/交互对齐的原型页，不代表最终 UI；字段命名遵循当前前端页面口径（例如里程碑父子关系字段为 `parentCodes`）。
